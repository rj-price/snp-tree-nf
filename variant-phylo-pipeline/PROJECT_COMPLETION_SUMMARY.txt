WHAT WAS BUILT
════════════════════════════════════════════════════════════════════════════════

A complete, production-ready bioinformatics pipeline that performs:

1. ✅ Quality Control (FastQC - raw and trimmed reads)
2. ✅ Read Trimming (Trimmomatic - adapters & quality)
3. ✅ Reference Genome Indexing (BWA + SAMtools)
4. ✅ Read Alignment (BWA-MEM with read groups)
5. ✅ BAM Processing (convert, sort, index, statistics)
6. ✅ Variant Calling (BCFtools mpileup + call)
7. ✅ SNP Filtering (quality, depth, mapping quality)
8. ✅ Format Conversion (VCF → PHYLIP + NEXUS)
9. ✅ Phylogenetic Tree Construction (RAxML with bootstraps)
10. ✅ Comprehensive Reporting (MultiQC)

USAGE SCENARIOS
════════════════════════════════════════════════════════════════════════════════

BASIC EXECUTION:
```bash
nextflow run main.nf \
    --input samples.csv \
    --fasta reference.fa
```

WITH OPTIONS:
```bash
nextflow run main.nf \
    --input samples.csv \
    --fasta reference.fa \
    --adapters TruSeq3-PE.fa \
    --outdir results \
    --max_cpus 16 \
    --max_memory 32.GB
```

RESUME CAPABILITY:
```bash
nextflow run main.nf \
    --input samples.csv \
    --fasta reference.fa \
    -resume
```

CUSTOM PARAMETERS:
```bash
nextflow run main.nf \
    --input samples.csv \
    --fasta reference.fa \
    --filter_qual 30 \
    --raxml_bootstraps 1000
```

HOW TO USE THIS PIPELINE
════════════════════════════════════════════════════════════════════════════════

1. READ DOCUMENTATION:
   Start with QUICK_START.md for immediate usage
   Review README.md for complete details
   Consult USAGE.md for specific scenarios

2. PREPARE INPUT:
   Create samplesheet.csv (see assets/samplesheet.csv)
   Obtain reference genome (FASTA format)
   Optional: Adapter sequences for trimming

3. RUN PIPELINE:
   nextflow run main.nf --input samples.csv --fasta reference.fa

4. REVIEW RESULTS:
   Check MultiQC report (results/multiqc/multiqc_report.html)
   View phylogenetic tree (results/phylogenetics/raxml/)
   Inspect filtered variants (results/variants/)

5. TROUBLESHOOTING:
   Review .nextflow.log for errors
   Check COMPLETE_PIPELINE_INFO.txt troubleshooting section
   Use -resume flag to restart from checkpoint
